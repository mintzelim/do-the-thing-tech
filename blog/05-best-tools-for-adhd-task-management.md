faq:
  - q: What makes a task management tool ADHD-friendly?
    a: >-
      An ADHD-friendly tool reduces cognitive load rather than adding to it.
      Look for tools with minimal setup, visual task breakdown, time estimation
      built in, and external cues (timers, reminders). Tools that add
      complexity, subscription fatigue, or require extensive setup often fail
      for ADHD brains.
  - q: Do ADHD-specific tools really work better than general ones?
    a: >-
      Yes, because they're designed around ADHD-specific challenges. General
      tools assume consistent executive function. ADHD-specific tools
      compensate for executive dysfunction by providing external structure,
      visual time cues, and initiation support.
  - q: Can free tools be effective for ADHD task management?
    a: >-
      Absolutely. DoTheThing is free and covers the core need: breaking tasks
      into micro-steps. Simple tools like a visual timer and a notebook can be
      just as effective as paid apps — the key is consistent use of whatever
      system you choose.
  - q: Should I use multiple tools or one comprehensive app?
    a: >-
      Fewer tools are better for ADHD brains. Each additional tool adds a
      friction point and a forgetting point. One tool that handles task
      breakdown, time estimation, and tracking is more ADHD-friendly than
      managing four different apps.
