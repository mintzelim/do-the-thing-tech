## Why ADHD Brains Struggle to Start Tasks

Dr. Russell Barkley, PhD, puts it plainly: "It is not that the individual does not know what to do. It is that somehow it does not get done."

[A 2024 review in Nature Reviews Psychology by Kofler et al.](https://pmc.ncbi.nlm.nih.gov/articles/PMC11485171/) confirms that executive function deficits, particularly in working memory, inhibitory control, and planning, are among the most consistently documented cognitive features of ADHD. The prefrontal cortex handles planning, sequencing, and task initiation, and it runs on dopamine, as confirmed by [Isaac, Lopez, and Escobar (2024) in Frontiers in Psychiatry](https://pmc.ncbi.nlm.nih.gov/articles/PMC10827919/). In ADHD brains, that dopamine supply is reduced.

The practical result: generating a plan for a multi-step task carries nearly the same cognitive cost as doing the task. Your brain faces the full weight of the project before you've taken a single action. That weight triggers avoidance. The task stays undone.

[CHADD identifies task initiation as 1 of the 6 core executive function clusters impaired by ADHD](https://chadd.org/for-adults/time-management-and-adhd-to-do-lists/), alongside focus, effort, emotion, working memory, and action. Task initiation specifically includes organising materials, estimating time, and getting started, everything that happens before any visible work begins.

This is why "just start" advice fails. Starting requires planning. Planning is what the brain can't do.

## What AI Task Breakdown Actually Does

An AI task breakdown tool takes a vague task and returns a specific, numbered, actionable sequence.

Input: "Prepare the Q3 client presentation."

Output:
1. Open last quarter's presentation and save a copy as Q3 draft (3 min)
2. List the 3 main updates since Q2 in a scratch doc (8 min)
3. Update slide 2 with the new revenue figure (5 min)
4. Update slide 4 with the new project status (5 min)

The transformation matters because it removes the 2 steps that stop ADHD brains before they start: figuring out what to do first, and estimating how long it will take. Once both are handled externally, the brain is no longer asked to plan. It's asked to execute 1 specific, bounded, timed action. That's a fundamentally smaller ask.

## The 3 Features That Make AI Breakdown ADHD-Specific

### Feature 1: ADHD Time Buffers

ADHD brains consistently underestimate how long tasks take, a symptom called time blindness. A plan built on optimistic estimates collapses by midday. An ADHD-specific tool adds a 20-30% buffer to every time estimate, so the planned duration lands closer to the actual one.

### Feature 2: Focus-Level Calibration

Your executive function capacity varies throughout the day. A plan built for a hyperfocused Tuesday morning will struggle on a distracted Wednesday afternoon.

An ADHD-specific tool adjusts the plan to your actual cognitive state. On a distracted day, steps are smaller and time estimates are longer. On a hyperfocused day, sessions can run deeper. The plan fits the brain you have today.

### Feature 3: Step Granularity Small Enough to Start

The test for a valid micro-step: can you do this right now, without deciding anything or preparing anything first?

"Write the introduction" still requires decisions. "Open the doc and type the first sentence" is ready to act on.

ADHD-specific tools calibrate to this standard. General AI tools often produce steps that still require planning to begin.
